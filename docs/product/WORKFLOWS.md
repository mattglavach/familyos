# Workflows

## Purpose
Workflows describe how Family OS should fit into real household routines. They are product guidance, not implementation steps.

## Morning Routine
Goal: Start the day with clarity.

Flow:
1. Open Home.
2. Review Today's Priorities.
3. Check Today's Schedule.
4. Complete or defer quick tasks.
5. Add anything remembered through Quick Add.

Release 1.0 status: Core.

## Daily Planning
Goal: Keep household responsibilities moving.

Flow:
1. Review task lists and assignments.
2. Create or update tasks.
3. Reassign work if needed.
4. Complete done items.
5. Check household context and calendar status.

Release 1.0 status: Core.

## School
Goal: Track school-related schedule, forms, tasks, and future planning.

Flow:
1. Review calendar events.
2. Track school tasks and deadlines.
3. Use future documents and college modules for deeper planning.

Release 1.0 status: Partial through calendar/tasks; deeper school workflows are future.

## Weekend
Goal: Coordinate chores, errands, pool, yard, home, and family plans.

Flow:
1. Review Home for weekend priorities.
2. Filter tasks by household member or category.
3. Add maintenance or shopping items.
4. Check calendar commitments.

Release 1.0 status: Partial through dashboard/tasks.

## Vacation
Goal: Plan travel tasks, documents, packing, pet/home care, and itinerary.

Release 1.0 status: Future.

Future flow:
1. Create trip.
2. Generate tasks and packing lists.
3. Attach documents.
4. Track calendar and reminders.

## Shopping
Goal: Maintain shared grocery and household purchase lists.

Release 1.0 status: Future.

Future flow:
1. Add items through Quick Add.
2. Group by store/category.
3. Assign ownership.
4. Check off during shopping.

## Pool Maintenance
Goal: Keep pool chemistry and maintenance on track.

Release 1.0 status: Future/partial.

Future flow:
1. Enter test readings.
2. Review what the pool needs and why it matters.
3. Confirm any recommended chemical or maintenance action.
4. Log treatments, equipment changes, weather context, parties, and water clarity.
5. Track trend, confidence, safety notes, and next test date.

Release 1.3.2 planning note: Pool Care Assistant is documented as a future Home Platform submodule in `docs/product/POOL_CARE_ASSISTANT.md`. Do not expose future Pool Quick Add targets or AI coaching until the Pool module is explicitly scoped.

## Home Maintenance
Goal: Track recurring home care and equipment history.

Release 1.0 status: Future.

Future flow:
1. Add home asset or maintenance item.
2. Set interval.
3. Receive reminder.
4. Log completion and notes.

## College Planning
Goal: Track long-term education decisions, savings, deadlines, and research.

Release 1.0 status: Future/partial.

Future flow:
1. Track milestones and deadlines.
2. Link tasks and documents.
3. Compare options.
4. Monitor savings and scholarships.

## Holiday Planning
Goal: Coordinate gifts, events, travel, meals, and household preparation.

Release 1.0 status: Future.

Future flow:
1. Create holiday plan.
2. Add gift and shopping lists.
3. Assign tasks.
4. Track dates and reminders.
