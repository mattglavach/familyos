# Success Metrics

## Source Of Truth
These metrics support the product vision in `docs/product/PRODUCT_VISION.md`.

## Usage Metrics
- Weekly active use
- Number of modules used
- Number of records added
- Number of reminders completed
- Number of household members actively participating
- Frequency of dashboard use

## Practical Value Metrics
- Fewer missed tasks
- Faster access to key information
- Better pool/home/finance tracking
- Fewer repeated manual calculations
- Faster morning planning
- More visible task ownership
- Fewer household handoffs lost in messages or memory

## Product Quality Metrics
- Mobile usability
- Page load speed
- Few bugs after release
- Consistent UI patterns
- Low friction for data entry
- Clear empty, loading, error, and success states
- Trustworthy permission behavior
