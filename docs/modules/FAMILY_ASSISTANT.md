# Family Assistant User Guide

Family Assistant helps answer “what needs attention?” and “what should we do next?” using the FamilyOS modules you choose. It can summarize, prioritize, explain, build a weekly plan, and draft actions.

It cannot independently create, edit, delete, complete, schedule, log, or dose anything. Proposed changes open an editable review in the owning FamilyOS workflow. Nothing changes until you explicitly save there.

Use the context control to include only relevant modules. FamilyOS excludes attachments, credentials, authentication tokens, database details, and unnecessary sensitive fields. Source labels show which modules informed the response. Confirmed facts come from FamilyOS, calculated observations come from deterministic rules, and suggestions are AI-generated.

Feedback controls suppress repetition and adjust recommendation preferences without training an external model or creating an opaque profile. Reset those preferences in Settings. Conversation history lasts only for the current on-screen session.

When AI is disabled, unavailable, or times out, FamilyOS continues to work and displays deterministic household facts. Preserve your question and retry only when useful.
