# Garden

## Purpose
The Garden module tracks plants, watering schedules, fertilizing, harvests, issues, photos, and seasonal notes.

## MVP Scope
- Summary view
- Add/edit records
- History list
- Basic reminders or notes
- Mobile-friendly layout

## Future Enhancements
- AI summaries
- Automated reminders
- Charts and trends
- Import/export
- Notifications

## Acceptance Criteria Template
- User can view module landing page.
- User can add a record.
- User can edit or delete where appropriate.
- Empty states are clear.
- Mobile layout works.
