# Tasks

Release 1.8.4 uses Due, My Tasks, All Tasks, and Completed as the primary views, with Due first and selected by default. Due contains incomplete overdue and today tasks using browser-local, date-only semantics. My Tasks includes direct assignments and shared or unassigned household responsibilities, but excludes tasks assigned exclusively to someone else, even when created by the signed-in user.

## Purpose
The Tasks module manages chores, recurring responsibilities, one-off tasks, reminders, assignments, priorities, and completion history.

## MVP Scope
- Summary view
- Add/edit records
- History list
- Basic reminders or notes
- Mobile-friendly layout

## Future Enhancements
- AI summaries
- Automated reminders
- Charts and trends
- Import/export
- Notifications

## Acceptance Criteria Template
- User can view module landing page.
- User can add a record.
- User can edit or delete where appropriate.
- Empty states are clear.
- Mobile layout works.
