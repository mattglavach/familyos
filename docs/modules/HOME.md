# Home

## Release 1.8 boundary

Pool remains the implemented Home Operations workspace. Its status flows into the Household Context Service and Dashboard without adding broad asset categories. Gardening remains unavailable and is scheduled for Release 1.9. Vehicles, appliances, tools, inventory, warranties, shopping, meal planning, smart-home control, and automatic Pool control are not added to Home.

Home manages recurring household operational systems that benefit from structured measurements, history, treatments, maintenance workflows, and ChatGPT context. Current roadmap ownership is limited to Pool, Garden, operational settings, histories, linked Tasks, and ChatGPT context. Vehicles, appliances, tools, inventory, warranties, projects, documents, and broad asset tracking are excluded.

## Purpose
The Home module tracks maintenance, equipment, warranties, projects, vendors, seasonal tasks, and household systems.

## MVP Scope
- Summary view
- Add/edit records
- History list
- Basic reminders or notes
- Mobile-friendly layout

## Future Enhancements
- AI summaries
- Automated reminders
- Charts and trends
- Import/export
- Notifications

## Acceptance Criteria Template
- User can view module landing page.
- User can add a record.
- User can edit or delete where appropriate.
- Empty states are clear.
- Mobile layout works.
