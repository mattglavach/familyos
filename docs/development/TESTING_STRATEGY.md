# Testing Strategy

## Manual Testing
Every feature should be checked for:
- Desktop layout
- Mobile layout
- Empty state
- Loading state
- Error state
- Invalid input
- Authenticated access

## Automated Testing
Add automated tests for:
- Business logic
- Calculations
- Data transformations
- Critical forms

## High-Risk Areas
- Finance calculations
- Retirement projections
- College planning assumptions
- Medical data
- Authentication and permissions
