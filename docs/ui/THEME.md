# Theme

## Theme Goals
- Calm
- Professional
- Family-friendly
- Modern

## Guidelines
- Use consistent spacing scale.
- Avoid excessive colors.
- Use icons consistently.
- Support light and future dark mode.
- Keep typography readable.

## Release 1.0.3 Tokens
- Colors are exposed as CSS variables and Tailwind tokens for foreground, background, card, primary, secondary, muted, accent, destructive, success, warning, and info states.
- Radius tokens are `--radius-sm`, `--radius-md`, and `--radius-lg`.
- Elevation tokens are `--shadow-soft` and `--shadow-card`.
- Motion tokens are `--duration-fast`, `--duration-normal`, and `--duration-slow`.
- New UI should use tokens instead of hard-coded colors, shadows, radii, or timing values.
