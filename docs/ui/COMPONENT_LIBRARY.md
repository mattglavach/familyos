# Component Library

## Shared Components
- AppShell
- Sidebar
- TopNav
- ModuleCard
- StatCard
- EmptyState
- LoadingState
- ErrorState
- ConfirmDialog
- DataTable
- FormSection
- ChartCard

## Rules
- Reuse shared components before creating new ones.
- Components should accept typed props.
- Keep business logic out of visual components when practical.
