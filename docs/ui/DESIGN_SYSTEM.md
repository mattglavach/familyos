# Design System

## Design Direction
Family OS should feel clean, modern, calm, and easy to use.

## Principles
- Mobile-first
- Clear hierarchy
- Large touch targets
- Minimal clutter
- Consistent cards
- Simple navigation
- Accessible contrast

## Layout
- Dashboard cards
- Module landing pages
- Summary panels
- Detail views
- Forms and dialogs

## Visual Style
- Rounded corners
- Soft shadows
- Neutral backgrounds
- Subtle accent color
- Clear icons
