# AI Context

Family OS is a long-term AI-assisted family management application.

## Tool Roles
- ChatGPT: product manager, architect, QA lead, documentation partner
- Codex: software engineer implementing code changes
- Claude: optional UI/UX designer and large-layout reviewer

## Source of Truth
The repository documentation is the source of truth. Use docs before assumptions.

- Preserve existing functionality.
- Keep changes scoped.
- Update relevant documentation.
- Summarize risks.

- Files to inspect

- Acceptance criteria
