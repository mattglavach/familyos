# AI Context

Family OS is a long-term AI-assisted family management application.

## Tool Roles
- ChatGPT: product manager, architect, QA lead, documentation partner
- Codex: software engineer owning complete engineering execution workstreams
- Claude: optional UI/UX designer and large-layout reviewer
- GitHub: shared source of truth

## Source of Truth
The repository documentation is the source of truth. Use docs before assumptions. Process docs in `docs/process/` govern release, feature, validation, review, git, and documentation workflows.

## Development Expectations
- Ask for clarification only when needed.
- Make pragmatic decisions.
- Preserve existing functionality.
- Keep changes scoped.
- Execute complete milestones rather than isolated tasks when a release or feature workstream is assigned.
- Update relevant documentation.
- Summarize risks.

## Preferred Output for Implementation Tasks
- Requirements
- Files to inspect
- Files to modify
- Acceptance criteria
- Testing checklist
- Documentation updates
