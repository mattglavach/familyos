# FamilyOS Version

Current version: **2.0.0**

Release date: 2026-07-12

Release 2.0.0 introduces the provider-neutral Context Engine, privacy-aware Prompt Builder, and review-first AI Workspace. FamilyOS remains the system of record: prompts are copied manually, AI responses are never stored automatically, and every identified suggestion requires explicit review. No database migration, provider authentication, or dependency addition is required.
